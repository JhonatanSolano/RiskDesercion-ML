# Preparaci-n-y-Construcci-n-de-Datos-para-el-An-lisis-de-Deserci-n-Universitaria
Proyecto orientado a la limpieza, transformación y estructuración de datos académicos por cohortes estudiantiles, generando variables clave que permiten caracterizar el ingreso, permanencia y deserción, dejando el dataset listo para el modelamiento predictivo.
